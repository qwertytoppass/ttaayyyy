# Статичный сайт + отзывы

Файлы сайта находятся в `website/`.

- Откройте `website/index.html` в браузере.
- Отзывы хранятся в `website/reviews.json`.

Чтобы обновить `reviews.json` из БД бота (файл `bot_data.sqlite3` в той же папке), используйте скрипт:

```bash
python export_reviews.py
```

Скрипт попробует найти таблицу отзывов по распространённым именам и экспортировать записи в `website/reviews.json`.

Чтобы оставить отзыв пользователю нужно зарегистрироваться через Telegram-бота, работающего в проекте. В `website/index.html` замените `your_bot_username` на реальное имя вашего бота (например `my_bot`).
